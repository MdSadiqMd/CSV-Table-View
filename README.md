## Usage

1. Open any CSV file
2. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
3. Type "CSV: Open Preview"
4. Enjoy your beautiful table!

## Requirements

- VS Code 1.80.0 or higher

## Extension Settings

- `csv.previewRowCount`: Number of rows to load (default: 10000)
- `csv.delimiter`: Delimiter to use (default: "auto")
- `csv.showRowNumbers`: Show row numbers (default: true)
